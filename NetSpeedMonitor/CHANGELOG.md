# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02-20

### Added
- 实时网速监控显示在菜单栏
- 毛玻璃效果菜单面板
- 应用下载速度排行（Top 3）
- 流量使用累计统计排行（Top 3）
- 开机自启动选项
- 显示/隐藏上传下载速度开关
- 重置流量统计功能
- macOS 风格应用图标
- 支持深色/浅色模式自适应
- 支持多进程应用合并显示（如 Trae CN Helper）

### Technical
- Swift 5.5 + AppKit
- 使用 nettop 监控网络流量
- NSCache 优化图标缓存
- 字体和属性缓存优化
